# Shubham Jatania - Modern Resume

A premium, highly-styled personal resume website built with modern web standards. This project showcases a sophisticated "cooler" design aesthetic, incorporating glassmorphism, smooth animations, and a responsive layout.

## ✨ Features

- **Modern Aesthetic**: A sleek dark theme using a curated palette of deep slates, cyans, and purples.
- **Dynamic Animations**: Smooth entrance effects for sections and interactive hover states for a responsive feel.
- **Glassmorphism**: Elegant UI components with translucent, blurred backgrounds and subtle borders.
- **Fully Responsive**: Optimized for viewing across all devices, from mobile phones to high-resolution desktops.
- **Semantic HTML**: Built with accessibility and SEO best practices in mind.
- **Zero Dependencies**: Pure HTML and CSS (inline) for maximum performance and portability.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Modern CSS3 (Variables, Flexbox, Grid, Animations)
- **Typography**: Outfit & Inter (via Google Fonts)
